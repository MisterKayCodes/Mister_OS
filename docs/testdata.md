Note 2 — "Programming Notes: FastAPI"


FastAPI Learning Notes
- FastAPI uses Pydantic for request validation
- SQLAlchemy is the ORM, works great with SQLite for local dev
- Always use --reload flag when developing
- CORS middleware must be configured if frontend is on a different port
- Use Depends() for dependency injection (like auth tokens)
- ChromaDB stores vector embeddings locally, no GPU needed
- RAG = Retrieval Augmented Generation. Query vectors then pass context to LLM
- Uvicorn is the ASGI server that serves FastAPI
Remember: never commit the .env file to GitHub!
